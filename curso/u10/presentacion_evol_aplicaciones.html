Aplicaciones nativas para la nube

  • La utilización de toda la funcionalidad de la nube, principalmente la elasticidad, implica un cambio drástico en la arquitectura del software hacia lo que se conoce como cloud applications
  • Alternativamente a la utilización de aplicaciones específicamente desarrolladas a funcionar en una nube se pueden aprovechar ciertas funcionalidades de cloud computing para aplicaciones "clásicas"
  • El objetivo es desarrollar aplicaciones que puedan adaptarse a las necesidades de cada momento ejecutándose en una infraestructura elástica como las aplicaciones de Google Facebook
